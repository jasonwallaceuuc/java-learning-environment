This website is designed to run along side and assist students currently studying the Professional Software Development 1 for MSc Professional Software Development. There are a total of 11 topics covering the modules followed in the first Java module of the course. Under each topic is a short topic test. After each group of three topics there is a longer more complex test in which a pass mark of 50% must be achieved.
